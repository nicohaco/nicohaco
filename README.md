## NicoHacoとは？
[NicoBox](https://itunes.apple.com/jp/app/nicobox-%E3%83%8B%E3%82%B3%E3%83%9C%E3%83%83%E3%82%AF%E3%82%B9/id421753493?mt=8)のPC版アプリです。  
NicoBoxと同様に、ニコニコ動画を音楽として聞くことに特化しています。
